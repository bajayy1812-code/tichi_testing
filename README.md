# Playwright Automation Testing Framework

## Overview

A professional automation testing framework built using Playwright and JavaScript to automate the Login and Signup workflows of the Tichi web application.

## Features

- Page Object Model (POM)
- Cross-browser testing
- HTML reporting
- Automatic screenshots on failure
- Console error monitoring
- Network/API monitoring
- Performance monitoring
- Winston logging
- Data-driven testing
- GitHub Actions workflow

## Tech Stack

- Playwright
- JavaScript
- Node.js
- Winston
- GitHub Actions

## Project Structure

```
pages/
tests/
utils/
fixtures/
reports/
screenshots/
logs/
```

## Run the project

```bash
npm install
npx playwright test
```

## View the report

```bash
npx playwright show-report
```
